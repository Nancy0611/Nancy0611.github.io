CSS垂直水平居中

发表于 2019-04-18 | 分类于 CSS整理

最近都没怎么写笔记，通过面试发现自己的基础掌握的还不是很扎实，呀懒人还是得做总结多做笔记才是，下次要用到才不用重新学哈哈哈

一、水平居中

1. 行内元素

  • 父元素是块级元素

    • 直接给父元素设置 text-align:center;

  • 父元素不是块级元素：

    1. 先将父元素设置为块级元素
    2. 再给父元素设置 text:align:center;

2. 块级元素

  • 方案1：分宽度定不定两种情况

    • 定宽度：需要谁居中，给其设 margin:0 aut0; 使盒子自己居中

    • 不定宽度：设置子元素为 display:inline-block;或display:inline; 即将其转换成为行内块级元素或行内元素，再给父元素设置 text-align:center;

  • 方案2：使用定位属性

    • 设置父元素为相对定位relative
    • 设置子元素为绝对定位absolute
    • 设置子元素的left为50%
    • 子元素向左移动自身元素宽度的一半

  • 方案3：使用flexbox布局（宽度定不定都可以）

    • 给待处理的块级父元素添加属性

二、垂直居中

1. 行内元素

  • 单行的行内元素：设置单行行内元素的行高等于盒子的高

  • 多行行内元素：给父元素设置display:table-cell;和vertical-align:middle;

2. 块级元素

  • 方案1：使用定位

    1. 设置父元素为相对定位
    2. 设置子元素为绝对定位
    3. 设置子元素的top为50%
    4. 子元素向上移动自身高度的一半

  • 方案2：flexbox布局（高度定不定都行）

    • 给待处理的块级元素的父元素添加属性display:flex;和align-items:center;

三、水平垂直居中

1. 已知宽度和高度的元素

  • 方案1：position+左上
    1. 父元素设置为相对定位position:relative;
    2. 子元素设置为绝对定位position:absolute;
    3. 子元素距上50%，距左50%
    4. margin-left和margin-top分别为元素宽高的一半
  • 方案2：position上下左右为0+margin:auto
    1. 父元素设置相对定位
    2. 子元素设置绝对定位
    3. 子元素top、right、bottom、left为0
    4. margin为auto

2. 未知宽度和高度的元素

  • 方案3：position+transform+元素未知宽度

    1. 父元素设置相对定位position:relative;
    2. 子元素设置绝对定位position:absolute;
    3. 子元素距上50%，距左50%
    4. 向左和向上移动自身元素宽高的50%
    • 注：其实就是用transform:translate(-50%,-50%)代替了方案1的margin，translate(x,y)括号里的值为百分数时，会以目前元素本身的宽高做参考，xy为正分别表示右下方向，为负即相反方向，也就是左、上

  • 方案4：flex布局
