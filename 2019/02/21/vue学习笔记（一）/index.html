vue学习笔记（一）

发表于 2019-02-21

1. 数据绑定

  • 在vuejs中变量在html中绑定标签是两个花括号
1.1 vuejs的v-model属性（双向数据绑定）
  • v-model指令在表单控件元素上创建双向数据绑定，它会根据控件类型自动选取正确的方法来更新元素
  • vuejs双向绑定并不能适用于所有的html表单，限制为input、select、textarea、components
1.2 vuejs的v-text属性
  • 在vuejs中可以使用v-text属性来绑定数据

2. 常用指令

2.1 v-html
  • v-html更新元素或变量的innerHTML，按普通html解析，和v-text的区别是在变量中的html标签会被浏览器解析，比如<br>会直接解析为换行语法
2.2 v-show
  • 根据条件解析元素display属性的真假，如果为真则display为block，假的话则为none在页面不显示
2.3 v-if
  • v-if、v-else、v-else-ifif三兄弟，根据表达式的值的真假条件渲染元素，真假条件可使用true、false或表达式
  • v-else不需要表达式，但是需要跟在v-if、v-else-if后面
2.4 v-for
  • 循环遍历
2.5 v-on
  • v-on给元素绑定事件，v-on可简写为@

4. 挂载点、模板和实例关系

  • Vue实例中el对应的标签就是挂载点，Vue只会处理挂载点下的内容，挂载点内部的内容就是模板内容，可把模板放在Vue实例的template属性里面

5. 计算属性、侦听器

  • 计算属性computed是指一个属性的值是根据其他数据项计算出来的值，只有依赖的属性发生变化才会重新计算，如果依赖的属性没变化则利用以前计算结果的缓存
  • 侦听器watch监听某一数据或计算属性发生了变化，一旦发生变化就可在侦听器里做业务逻辑
  • methods和（watch/computed）对比：前者是需要手动调用才能执行；后者是以vue的依赖追踪机制为基础的，当某一个依赖数据发生变化的时候，所有依赖这个数据的“相关”数据“自动”发生变化，也就是自动调用相关的函数去实现数据的变动
  • computed和watch对比：前者是计算属性，在使用上和data对象里的数据属性是用一类的，擅长处理一个数据受多个数据影响；后者是监听机制+事件机制，擅长处理一个数据影响多个数据
